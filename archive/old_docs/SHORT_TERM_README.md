# 단기 투자 종목 선정 시스템

symposium V2(중장기 스윙)를 **단기 투자(3~10일 보유)** 용으로 변형한 한국 종목 스크리너

---

## 핵심 차이점

### V2 중장기 vs 단기

| 요소 | V2 중장기 | 단기 전략 |
|------|----------|----------|
| **보유 기간** | 1개월+ | 3~10일 |
| **팩터 가중치** | 밸류30/실적25/테마20/진입15/수급10 | 밸류10/실적15/테마25/진입35/수급15 |
| **테마 지속성** | 14일 | **5일** (빠른 회전) |
| **수급 기준** | 5일 순매수 | **3일 연속** 순매수 |
| **RSI 기간** | 14일 | **9일** (민감도↑) |
| **이평선** | 20/60일 | **5/20일** |
| **최소 시총** | 1,000억원 | **500억원** (중소형주 포함) |
| **최소 점수** | 45점 | **55점** (더 엄격) |
| **결과 개수** | TOP 30 | **TOP 10~15** |

---

## 팩터 구성 (100점 만점)

### 1. 밸류에이션 (10점) ↓
- **V2는 30점**: 중장기는 저평가가 중요
- **단기는 10점**: 단기에는 모멘텀이 더 중요
- PEG < 0.5 저평가면 가점
- Forward PE < 15 저평가면 가점

### 2. 실적 (15점) ↓
- **V2는 25점**: 26E/27E 컨센서스 중심
- **단기는 15점**: 직전 분기 서프라이즈 중심
- 전분기 EPS 20%+ 서프라이즈: 15점
- 전분기 EPS 10%+ 양호: 13점

### 3. 테마 (25점) ↑
- **V2는 20점**: 14일 지속성 + LLM 구조성 판정
- **단기는 25점**: 5일 지속성 + 당일 TOP 테마
- 5일 중 4일 이상 상승: 20점
- 당일 등락률 TOP 5 테마: +3점
- 순환매 감지 테마: +5점

### 4. 진입 타이밍 (35점) ↑↑ **가장 중요!**
- **V2는 15점**: RSI/볼린저 기본
- **단기는 35점**: 매수/과열 신호 정밀 분석

#### 매수 신호 (가점)
- RSI(9일) 30~40 과매도 반등: +10점
- 볼린저밴드 하단 터치 후 반등: +8점
- 5일선 골든크로스: +7점
- 눌림목 패턴 (1주 -3~-10% 조정 후 반등): +5점
- 거래량 2배+ 폭증: +5점

#### 과열 신호 (감점)
- RSI > 70: -10점
- 1주 급등 +30%+: -5점
- 볼린저밴드 상단 돌파: -8점

### 5. 수급 (15점) ↑
- **V2는 10점**: 5일 순매수
- **단기는 15점**: 3일 연속 순매수
- 외국인+기관 3일 연속 동반 순매수: 15점
- 외국인 or 기관 3일 연속: 10점
- 당일 동반 순매수: 8점

## 실행

### 기본 실행
```bash
python run_short_term.py
```
- TOP 15 종목 선정
- 최소 55점 필터
- **텔레그램 자동 전송** (설정 시)
- 결과: `symposium/screener/data/short_term/YYYY-MM-DD.json`

### 옵션
```bash
python run_short_term.py --top 10          # TOP 10만
python run_short_term.py --score 60        # 최소 60점으로 강화
python run_short_term.py --workers 16      # 병렬 처리 증가
python run_short_term.py --compact         # 텔레그램 간단 요약 (TOP 5)
python run_short_term.py --no-telegram    # 텔레그램 전송 안함
```

### 텔레그램 알림 설정
자세한 내용: [TELEGRAM_SETUP.md](../TELEGRAM_SETUP.md)

```bash
# 1. .env 파일에 추가
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id

# 2. 테스트
cd symposium/screener/short_term
python telegram_notifier.py

# 3. 실행
python run_short_term.py
# → 텔레그램으로 결과 자동 전송!
```

## 사용 시나리오

### 1. 매일 장 시작 전 (09:00)
```bash
python run_short_term.py --top 10 --score 60
```
- 전날 종가 기준으로 TOP 10 추출
- 높은 점수 기준으로 엄격 필터링

### 2. 장중 모니터링
- **진입 타이밍 35점**이 핵심
- RSI 과매도 구간 + 볼린저 하단 + 거래량 폭증 조합
- 5일선 골든크로스 확인

### 3. 보유 기간
- **목표**: 3~7일
- **손절**: -5~7% (감점 신호 발생 시 즉시)
- **익절**: +15~25% 또는 과열 신호 발생 시

### 4. 리밸런싱
- **매일 실행**: 순위 변동 확인
- **신규 진입**: 순환매 테마 교체 시
- **청산**: 과열 신호 (RSI>70, 1주 +30%+)

## 주의사항

### 1. 백테스트 필수
- 과거 6개월 데이터로 시뮬레이션
- 보유기간별 수익률 분석 (3/5/7일)
- 승률 / MDD / 샤프비율 측정

### 2. 리스크 관리
- **손절 규칙**: -5~7% 자동 손절
- **포지션 크기**: 종목당 5~10% (10~20종목 분산)
- **중소형주 슬리피지**: 체결가 불리할 수 있음

### 3. 시장 상황
- **강세장**: 진입 타이밍 35점 활용, 눌림목 매수
- **약세장**: 현금 비중 증가, 최소 점수 65점으로 상향
- **박스장**: 볼린저밴드 하단/상단 단타

### 4. 뉴스 필터
- 악재 발생 종목 즉시 제외
- 실적 발표일 D-7 이내 종목 주의
- 공시 모니터링 (유상증자, 경영권 분쟁 등)

## V2와의 병행 사용

### 중장기 (V2) + 단기 병행 전략
1. **V2 (symposium)**: 한국 TOP 30 중 장기 보유 (1~3개월)
2. **단기**: TOP 10~15 중 스윙 (3~7일)
3. **비중**: 중장기 60% / 단기 40%

### 시너지
- V2에서 발굴한 테마를 단기에서 빠르게 캐치
- 단기 눌림목 → V2 중장기 진입 타이밍 활용
- 순환매 감지 → 양쪽 모두 활용
