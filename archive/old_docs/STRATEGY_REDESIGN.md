## 1️⃣ 데이 트레이딩 (당일)

### 🎯 목표
- **보유**: 당일 매수 → 당일 매도
- **수익**: +2~5%
- **방식**: **당일 차트 보고 실시간 판단**

### ⏰ 타임라인

```
08:30 - 시초가 확인
  ↓
09:00 - 장 시작, 5~10분 관찰
  ↓
09:10 - 실시간 차트 보고 판단
  • 갭상승 후 조정 → 매수
  • 갭하락 후 반등 → 매수
  • 강한 상승 추세 → 추격 매수
  ↓
14:50 - 청산 준비
15:20 - 전량 청산
```

### 📊 판단 기준 (당일 차트)

#### 매수 신호
```
✅ 시초가 갭상승 → 09:10 조정 → 매수
✅ 시초가 갭하락 → 09:10 반등 → 매수
✅ 보합 개장 → 5분봉 강한 상승 → 추격
✅ 대형주 + 거래량 급증 → 매수
```

#### 회피 신호
```
❌ 시초가 급등(+5%) → 고점 위험
❌ 개장 후 5분봉 3연속 하락
❌ 거래량 미약
❌ 장중 변동성 과도
```

### 🛠️ 도구

#### 수동 판단 (추천)
- **차트 소프트웨어**: 키움 HTS, 이베스트 HTS
- **실시간 모니터링**: 5분봉 + 거래량
- **직접 판단**: 차트 패턴 + 체결 강도

#### 보조 스크립트
```python
# 09:00~09:10 실시간 데이터 수집
python monitor_morning_momentum.py

출력:
━━━━━━━━━━━━━━━━━━━━━━━━
09:05 현재 상황
━━━━━━━━━━━━━━━━━━━━━━━━
삼성전자
  시초가: +2.3% 갭상승
  현재: +1.8% (조정 중)
  거래량: 평균 대비 3.2배
  → 💡 매수 타이밍 접근

SK하이닉스
  시초가: +5.8% 급등
  현재: +6.2% (추가 상승)
  거래량: 평균 대비 5.1배
  → ⚠️ 과열, 관망
```

## 2️⃣ 단기 스윙 (3~7일) - 지금 만든 것

### 🎯 목표
- **보유**: 3~7일
- **수익**: +5~15%
- **방식**: **전날 데이터 기반 자동 분석**

### ⏰ 타임라인

```
매일 07:50 - 자동 실행
  ↓
TOP 10 종목 선정
  ↓
텔레그램 수신
  ↓
09:00~10:00 - 시초가 확인 후 진입
  ↓
3~7일 보유
  ↓
목표가 or 손절가 도달 시 청산
```

### 📊 전략

#### 현재 시스템 (백테스트 검증)
- ✅ 3일 연속 하락 → 반등 (+1.49%p)
- ✅ RSI 과매도 → 반등 (+0.54%p)
- ✅ 볼린저 하단 → 반등 (+0.70%p)

#### 한계
- ❌ 섹터 붕괴일 대응 불가
- ❌ 추세장 대응 부족
- ⚠️ 대형주 우대 필요

#### 개선 방향
```python
# 시장 상황 감지 추가
if KOSPI < -2%:
    return []  # 관망

# 섹터 분산
max_per_sector = 3

# 대형주 우대
if 시총 > 10조:
    score += 10
```

## 🎯 사용자 맞춤 추천

### 당신의 선택
> "데이 트레이딩은 그날 차트 보고 하고 싶다"

→ **완벽한 선택!**

### 이유
1. **전날 데이터 한계 명확**
   - 오늘처럼 급변하는 날 대응 불가
   - 추세 vs 역발상 판단 어려움

2. **차트가 정확**
   - 실시간 수급 반영
   - 체결 강도, 거래량 즉시 확인
   - 패턴 직접 판단

3. **백테스트 불가능**
   - 장중 판단 = 경험 + 직관
   - 자동화 어려움
