## 🎯 필요한 개선 사항

### 1. **시장 상황 필터** (최우선!)

```python
def check_market_safety():
    """
    시장 전체 상황 체크
    
    제외 조건:
    1. KOSPI 전일 -2% 이상 하락
    2. 변동성 지수(VIX) 급등
    3. 섹터 지수 -5% 이상
    """
    kospi = yf.Ticker("^KS11")
    df = kospi.history(period="5d")
    
    ret_1d = (df['Close'].iloc[-1] / df['Close'].iloc[-2] - 1) * 100
    
    if ret_1d < -2:
        return False, "시장 전체 하락, 관망"
    
    return True, "OK"
```

### 3. **섹터 분산 강제**

```python
def diversify_sectors(picks):
    """
    한 섹터 최대 3개 (30%)
    """
    sector_count = {}
    result = []
    
    for pick in picks:
        sector = get_sector(pick)
        if sector_count.get(sector, 0) < 3:
            result.append(pick)
            sector_count[sector] = sector_count.get(sector, 0) + 1
    
    return result
```

### 4. **급락 강도 구분**

```python
# 현재: 5일 -10% = 20점
# 문제: -10% vs -30% 구분 없음

# 개선:
if ret_5d < -15:  # 극단 급락

        score = 5   # 위험
```

## 💡 근본적 해결책

### 백테스트 한계 인정

```
백테스트 = 평균적인 날 학습
극단적인 날 = 예측 불가

→ 보수적 전략 필요
```

### 보수적 접근

1. **의심스러우면 제외**
   - 20일 -15% 이상: 제외
   - 섹터 집중: 제외
   - 시장 약세: 관망

2. **소액 분산**
   - 종목당 5~10%
   - 최소 10개 분산

3. **빠른 손절**
   - 당일 -3%: 손절 고려
   - -5%: 무조건 손절
