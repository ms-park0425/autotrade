## 개요

**매일 아침 7시 50분** 자동으로:
1. 전날 종가까지 데이터 분석
2. 오늘 오를 가능성 높은 종목 TOP 10 선정
3. 텔레그램 자동 전송

## 📊 점수 시스템 (개선됨)

### 총 100점 만점

| 팩터 | 배점 | 핵심 포인트 |
|------|------|------------|
| **진입 타이밍** | 35점 | RSI 과매도 반등, 눌림목 |
| **테마** | 25점 | 3일 지속성, 전일 HOT 테마 |
| **수급** | 15점 | **전일 동반 순매수** (가중) |
| **실적** | 15점 | 직전 분기 서프라이즈 |
| **밸류에이션** | 10점 | PEG, Fwd PE |

### 🎯 "오늘 오를 종목" 예측 강화

#### 1. 전일 수급 가중치 상향
```
전일 동반 순매수: 8점 → 10점 (강화)
3일 연속: 15점 → 13점 (하향)

이유: 전날 매수세가 당일 연결될 확률 높음
```

#### 2. 과열 필터 강화
```
RSI 기준: 70 → 65 (더 엄격)
볼린저 상단: -8점 → -12점 (강화)
3일 연속 +5%: -10점 (신규)

이유: 단기는 고점 진입 위험 높음
```

#### 3. 테마 신선도 체크
```
3일 지속성 기준 (5일→3일 단축)
전일 HOT 테마: +3점 → +5점 (강화)
7일+ 지속: -3점 (이미 늦음)

이유: 단기 테마는 빠르게 회전
```

#### 4. 진입 베이스 하향
```
베이스 점수: 8점 → 5점

이유: 타이밍 신호 없으면 대기
```

## ⚠️ 주의사항

### 1. 손절 규칙 필수
```
-5~7% 자동 손절
이유: 단기는 손절이 생명
```

### 2. 분산 투자
```
1종목당 최대 20% (5종목 이상)
이유: 단기 변동성 높음
```

### 3. 시초가 확인
```
갭상승 +3% 이상: 관망
갭하락 -3% 이하: 관망
보합권 진입: 추천

이유: 극단 갭은 당일 반전 위험
```

### 4. 뉴스 체크
```
악재 발생 시 즉시 제외
이유: 기술적 신호보다 뉴스 우선
```

## 🔧 문제 해결

### "필터 통과 종목 없음"
→ 정상 (시장 약세장 or 과열장)
→ 최소 점수 낮추기: `--score 55`

→ `.env` 파일 확인:
```
TELEGRAM_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
```

### "모듈 없음 에러"
→ 의존성 설치:
```bash
pip install -r requirements.txt
```
