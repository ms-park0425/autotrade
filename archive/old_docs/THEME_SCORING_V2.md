## 📊 테마 점수 (25점)

### 1️⃣ 테마 추세 점수 (20점)

#### ✅ 급락 후 반등 (최우선!)

| 조건 | 점수 | 백테스트 근거 |
|------|------|--------------|
| **5일 -10% 이하 급락** | **+20** | +2.40%, 승률 66% 🥇 |
| **5일 -10~-5% 하락** | **+15** | +0.55%, 승률 52% 🥈 |
| **5일 -5~0% 약하락** | **+10** | +0.44%, 승률 53% |
| 5일 0~5% 약상승 | +5 | +0.17% (미약) |
| 5일 +5~10% 상승 | +8 | +0.46% |
| **5일 +10% 이상 급등** | **-10** | -0.15% ❌ |

#### ⚠️ 단, 예외 조건 (필터링)

```python
# 극단적 조정은 제외
if ret_20d < -30:  # 1개월 -30% 이상
    score -= 15
    reason = "섹터 붕괴 위험"

if ret_20d < -20 and 섹터_동반_하락:
    score -= 10
    reason = "섹터 전체 약세"
```

## 🔧 구현 코드

```python
def calc_theme_score_v2(ticker_data, theme_name):
    """
    테마 점수 V2 - 백테스트 검증됨
    
    핵심: 약세 = 반등 기회!
    """
    score = 0
    signals = []
    
    # 5일 수익률
    ret_5d = ticker_data['ret_5d']
    ret_20d = ticker_data['ret_20d']
    
    # ─── 1. 추세 점수 (메인) ───
    if ret_5d < -10:
        score += 20
        signals.append(f"{theme_name} 급락 반등")
    elif ret_5d < -5:
        score += 15
        signals.append(f"{theme_name} 하락 반등")
    elif ret_5d < 0:
        score += 10
        signals.append(f"{theme_name} 약세 반등")
    elif 0 <= ret_5d < 5:
        score += 5
        signals.append(f"{theme_name} 보합")
    elif 5 <= ret_5d < 10:
        score += 8
        signals.append(f"{theme_name} 상승세")
    elif ret_5d >= 10:
        score -= 10
        signals.append(f"{theme_name} 급등 조정 위험")
    
    # ─── 2. 극단 필터 (페널티) ───
    if ret_20d < -30:
        score -= 15
        signals.append("섹터 붕괴 위험")
    elif ret_20d < -20:
        # 섹터 전체 하락 체크 필요
        score -= 10
        signals.append("섹터 전체 약세")
    
    # ─── 3. 보너스 ───
    # (순환매, HOT 테마 등)
    
    return max(0, min(25, score)), ", ".join(signals)
```

## ⚠️ 주의사항

### 1. 개별 약세 vs 섹터 붕괴 구분

```python
# 개별 약세 = 기회
if ret_5d < -10 and ret_20d > -20:
    score += 20  # OK

# 섹터 붕괴 = 위험
if ret_5d < -10 and ret_20d < -30:
    score = 0  # 제외
```

### 2. 시장 전체 하락 시

```python
if KOSPI_5d < -5:  # 시장 전체 하락
    # 약세 점수 절반
    score = score * 0.5
```

### 3. 손절 필수

```
약세 반등 전략 = 고위험
→ 손절 -5% 엄수!
```
