## 1️⃣ 데이 트레이딩 (Day Trading)

### 🎯 목표
- **보유 기간**: 당일 매수 → 당일 매도
- **목표 수익률**: +2~5%
- **손절**: -2%

### 📊 중점 요소 (100점)

| 팩터 | 배점 | 설명 |
|------|------|------|
| **당일 모멘텀** | 40점 | 시초가 갭, 장초반 거래량, 체결 강도 |
| **기술적 타이밍** | 30점 | RSI, 볼린저, 전일 패턴 |
| **테마 핫도** | 20점 | 당일 테마 순위, 뉴스 |
| **요일 효과** | 10점 | 화요일 +10, 금요일 -10 |

### 🔑 핵심 신호 (백테스트 검증)

#### ✅ 강력 매수 신호
1. **3일 연속 하락** (+1.49%p) 🥇
2. **장대음봉 + 당일 갭상승** (+0.7%p)
3. **전일 저가 근접 마감** (+0.40%p)
4. **화요일** (+0.28%p)

#### ❌ 회피 신호
1. **전일 장대양봉** (-0.31%p)
2. **전일 고가 마감** (-0.23%p)
3. **금요일** (-0.54%p)
4. **갭상승 +3% 이상** (조정 위험)

### 📅 실행 스케줄

```bash
07:50 - 후보 선정 (전날 데이터 분석)
08:30 - 시초가 확인
09:00 - 매수 진입 (상위 5종목)
14:50 - 익절/손절 확인
15:20 - 당일 청산
```

### 📁 파일 구조

```
autotrade/
├── day_trading/
│   ├── scorer_day.py          # 데이 트레이딩 점수
│   ├── realtime_monitor.py    # 장중 실시간 모니터링
│   ├── entry_timer.py          # 진입 타이밍 (09:00~09:30)
│   └── exit_manager.py         # 청산 관리 (14:50~15:20)
│
└── run_day_trading.py          # 메인 실행 스크립트
```

### 🎯 점수 예시

```python
데이 트레이딩 점수 (100점)
━━━━━━━━━━━━━━━━━━

  - 3일 연속 하락 전일 마감 +15
  - 시초가 갭상승 +1.2% +10
  - 장초반 거래량 1.5배 +10
  
기술적: 28/30
  - RSI 35 과매도 +10
  - 볼린저 하단 반등 +8
  - 전일 저가 근접 마감 +10

테마: 18/20
  - AI반도체 당일 1위 +15
  - 전일 HOT 지속 +3

요일: 10/10
  - 화요일 +10

━━━━━━━━━━━━━━━━━━
총점: 91점 → 적극 매수!
```

- **손절**: -5%

| **테마** | 25점 | 3일 지속성, 순환매 |
| **수급** | 15점 | 전일 동반 순매수 |
| **실적** | 15점 | 전분기 서프라이즈 |

### 🔑 핵심 신호

현재 시스템과 동일 (SCORING_SYSTEM.md 참조)

### 📅 실행 스케줄

## 3️⃣ 중장기 투자 (Long-term)

### 🎯 목표
- **보유 기간**: 2~6개월
- **목표 수익률**: +20~50%+
- **손절**: -15%

### 📊 중점 요소 (100점)

| 팩터 | 배점 | 설명 |
|------|------|------|
| **구조적 테마** | 30점 | 14일+ 지속성, AI/반도체/바이오 |
| **밸류에이션** | 25점 | PEG, PER, 저평가 |
| **실적 성장** | 25점 | 연간 성장률, 가이던스 |
| **수급 추세** | 10점 | 외국인 누적 매수 |
| **진입 타이밍** | 10점 | 눌림목 진입 |

### 🔑 핵심 신호

#### ✅ 강력 매수 신호
1. **구조적 성장 테마** (AI, 반도체, 신재생)
2. **PEG < 0.5** (저평가 성장주)
3. **연간 실적 +30%+**
4. **외국인 3개월 누적 순매수**
5. **단기 조정** (20일선 터치)

#### ❌ 회피 신호
1. **PEG > 2.0** (고평가)
2. **실적 가이던스 하향**
3. **구조적 테마 붕괴**

### 📅 실행 스케줄

```bash
매주 일요일 21:00 - 자동 실행
      → TOP 30 선정
      → 텔레그램 전송

월 1회 - 포트폴리오 리밸런싱
분기 1회 - 실적 시즌 리뷰
```
