## 진입 타이밍 상세 (단기의 핵심)

### V2 중장기 (15점)
```
베이스 8점
- 눌림목: +5
- RSI 35~50: +3
- 볼린저 하단: +3
- RSI > 70 과열: -5
- 1주 급등 > 15%: -5
```

### 단기 (35점) - 2배 이상 강화!
```
베이스 8점

[매수 신호]
- RSI(9일) 30~40 과매도 반등: +10점
- 볼린저밴드 하단 터치 후 반등: +8점
- 5일선 골든크로스: +7점
- 눌림목 (1주 -3~-10% + 반등): +5점
- 거래량 2배+ 폭증: +5점

[과열 신호]
- RSI > 70: -10점
- 볼린저 상단 돌파: -8점
- 1주 급등 +30%+: -5점
```

## 사용 시나리오

### V2 중장기
```python
# 매일 17:00 실행
python symposium/run_v2.py

# 결과: 한국 TOP 30 + 미국 TOP 30
# 보유: 1~3개월
# 목표: +30~50%
# 손절: -15%
```

### 단기
```python
# 매일 장 시작 전 09:00
python run_short_term.py --top 10 --score 60

# 결과: 한국 TOP 10~15
# 보유: 3~7일
# 목표: +15~25%
# 손절: -5~7% (빠른 손절)
```

## 장단점

### V2 중장기
**장점**
- 구조적 성장주 발굴
- 큰 수익 가능 (+30~50%)
- 적은 거래 비용
- 심리적 안정

**단점**
- 긴 보유 기간
- 단기 변동성 노출
- 자금 회전율 낮음

### 단기
**장점**
- 빠른 자금 회전
- 타이밍 포착 정밀
- 손절 빠름 (손실 제한)
- 다양한 기회 포착

**단점**
- 거래 비용 높음
- 집중력 필요
- 슬리피지 (중소형주)
- 과최적화 위험
