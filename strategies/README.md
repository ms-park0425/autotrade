# 📊 AutoTrade 투자 전략 시스템

## 폴더 구조

```
strategies/
├── long_term/          # 장기 투자 (수개월~수년)
├── swing/              # 스윙 투자 (1주일 내외)
└── intraday/           # 당일 매매 (9시~10시 차트)
```

---

## 1️⃣ Long Term (장기 투자)

**목표**: 앞으로 몇 개월~수년간 오를 종목
**보유기간**: 3개월~수년
**기준**: 펀더멘털 + 구조적 테마 + 밸류에이션

### 핵심 지표
- ✅ 볼린저밴드 (기술적 진입점)
- ✅ 수급 (기관/외국인 순매수)
- ✅ 실적 (EPS 성장, 서프라이즈)
- ✅ 밸류에이션 (PER, PEG)
- ✅ 구조적 테마 (AI반도체, 2차전지, 방위산업 등)

### 실행
```bash
# 장기 투자 후보 스캔 (TOP 20)
python strategies/long_term/run_scan.py

# 자동 스케줄 (주 1회 일요일 오후 8시)
python strategies/long_term/run_scan.py --schedule
```

### 점수 체계 (100점)
- 펀더멘털: 40점 (실적 25 + 밸류 15)
- 테마: 30점 (구조적 지속성)
- 기술적: 20점 (진입 타이밍)
- 수급: 10점 (장기 누적)

---

## 2️⃣ Swing (단기 스윙)

**목표**: 일주일 내 단기 수익
**보유기간**: 3일~1주일
**기준**: 기술적 신호 + 단기 테마 + 수급

### 핵심 지표
- ✅ RSI 과매도 반등 (35~45)
- ✅ 눌림목 패턴
- ✅ 전일 기관/외국인 순매수
- ✅ 단기 HOT 테마 (3일 지속성)
- ✅ 거래량 급증

### 실행
```bash
# 전날 밤 or 당일 아침 7:50 - 기술적 후보 선정
python strategies/swing/run_scan.py

# 당일 아침 8:00~8:50 - 프리마켓 스캔
python strategies/swing/run_premarket.py --schedule
```

### 점수 체계 (100점)
- 진입 타이밍: 35점 (RSI, 거래량)
- 테마: 25점 (단기 모멘텀)
- 수급: 20점 (전일 순매수 가중)
- 실적: 15점
- 프리마켓: +15점 (갭상승 보너스)

---

## 3️⃣ Intraday (당일 매매)

**목표**: 오늘 9시~10시 차트로 당일 매도
**보유기간**: 수십분~수시간
**기준**: 실시간 차트 패턴 + 거래량 폭증

### 핵심 지표
- ✅ 9:00~10:00 급등 패턴
- ✅ 거래량 5배 이상 폭증
- ✅ 갭상승 후 지속 상승
- ✅ 신규 테마 출현
- ✅ 호가창 매수 우위

### 실행
```bash
# 실시간 모니터링 (9:00 시작)
python strategies/intraday/run_monitor.py

# 10:00 시점 스캔 (1시간 패턴 분석)
python strategies/intraday/run_scan_10am.py
```

### 점수 체계 (100점)
- 차트 패턴: 40점 (급등 지속성)
- 거래량: 30점 (전일 대비 배수)
- 호가창: 20점 (매수/매도 비율)
- 테마: 10점 (당일 신규 테마)

## 🎯 전략 선택 가이드

| 상황 | 추천 전략 | 이유 |
|------|----------|------|
| 직장인, 시간 부족 | Long Term | 매매 빈도 낮음 |
| 매일 확인 가능 | Swing | 적당한 빈도 |
| 전업 트레이더 | Intraday | 실시간 모니터링 |
| 안정 추구 | Long Term | 펀더멘털 기반 |
| 수익률 추구 | Swing + Intraday | 고위험 고수익 |

## ⚙️ 설정

각 전략은 독립된 설정 파일:
- `strategies/long_term/config.json`
- `strategies/swing/config.json`
- `strategies/intraday/config.json`

```
TELEGRAM_TOKEN=your_bot_token
TELEGRAM_CHAT_ID_LONG=your_long_chat_id
TELEGRAM_CHAT_ID_SWING=your_swing_chat_id
TELEGRAM_CHAT_ID_INTRA=your_intra_chat_id
```
