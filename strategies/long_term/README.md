# 📈 Long Term Strategy (장기 투자)

## 개요

**목표**: 앞으로 몇 개월~수년간 성장할 종목 선정
**보유기간**: 3개월~수년
**리밸런싱**: 주 1회 (일요일 저녁)

---

## 핵심 철학

### 1. 구조적 테마 중심
- ❌ 단기 테마 (정책, 이벤트)
- ✅ 구조적 테마 (AI반도체, 2차전지, 방위산업)
- **이유**: 장기적으로 수요 증가가 확실한 분야

### 2. 펀더멘털 우선
- EPS 성장률 (최근 2분기)
- 실적 서프라이즈 (컨센서스 대비)
- 밸류에이션 (PER, PEG)

### 3. 기술적 진입점
- 볼린저밴드 하단 (저평가 구간)
- RSI 과매도 (30~40)
- 장기 이평선 지지 (120일, 200일)

### 4. 수급 확인
- 기관/외국인 누적 순매수 (3개월)
- 대량 매집 흔적

---

## 점수 체계 (100점)

### 펀더멘털 (40점)
```python
실적 점수 (25점):
  - EPS 성장률: 15점 (YoY, QoQ)
  - 실적 서프라이즈: 10점 (컨센서스 대비)

밸류에이션 점수 (15점):
  - PEG Ratio: 10점 (성장 대비 저평가)
  - Forward PE: 5점 (미래 실적 반영)
```

### 테마 점수 (30점)
```python
구조적 테마 매칭:
  - 핵심 테마 (AI반도체, 2차전지 등): 25점
  - 테마 지속성 (6개월+): +5점
  - 단기 테마 (정책, 이벤트): 10점 (하향)
```

### 기술적 점수 (20점)
```python
진입 타이밍:
  - 볼린저밴드 하단: 8점
  - RSI 과매도 (30~40): 7점
  - 이평선 지지 (120일/200일): 5점
```

### 수급 점수 (10점)
```python
장기 수급:
  - 3개월 기관 순매수: 5점
  - 3개월 외국인 순매수: 5점
```

## 필터 기준

### 최소 요건
```python
✅ 통과 조건
- 시총: 5,000억원 이상
- 일평균 거래대금: 50억원 이상
- 최소 점수: 70점 이상
- EPS 성장률: 10% 이상

❌ 제외 조건
- 적자 기업
- PER > 50 (극고평가)
- 부채비율 > 200%
- 단기 급등 (1개월 +30% 이상)
```
