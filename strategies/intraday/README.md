# 🔥 Intraday Strategy (당일 매매)

## 개요

**목표**: 오늘 9시~10시 차트로 당일 매도
**보유기간**: 수십분~수시간 (장 마감 전 청산)
**수익 목표**: +3~7% (빠른 회전)

---

## 핵심 철학

### 1. 속도가 생명
- ⚡ 9:00 시초가 → 10:00 스캔 (1시간)
- ⚡ 급등 알림 즉시 확인
- ⚡ 의사결정 3분 이내

### 2. 차트 패턴 우선
- 갭상승 후 지속 상승
- V자 반등
- 거래량 5배 이상 폭증
- 매수 호가 압도

### 3. 테마 신선도
- 당일 신규 테마 출현
- 뉴스/공시 직후
- 커뮤니티 버즈 급증

### 4. 손익 관리 철저
- 익절: +5~7% 즉시
- 손절: -2~3% 무조건
- 보유 시간: 최대 3시간

## 점수 체계 (100점)

### 차트 패턴 (40점)
```python
급등 지속성:
  - 9:00~10:00 +5% 이상: 20점
  - 조정 없이 지속 상승: 15점
  - V자 반등 패턴: 12점
  - 갭상승 후 상승: 10점

분봉 패턴:
  - 5분봉 연속 양봉: 10점
  - 상승 추세선 유지: 8점
  - 이평선 정배열: 5점
```

### 거래량 (30점)
```python
전일 대비:
  - 5배 이상: 25점
  - 3배 이상: 18점
  - 2배 이상: 12점
  - 1.5배 이상: 5점

회전율:
  - 10% 이상: +5점
  - 시가총액 대비 고회전
```

### 호가창 (20점)
```python
매수/매도 비율:
  - 매수 70% 이상: 15점
  - 매수 60~70%: 10점
  - 매수 50~60%: 5점

호가 잔량:
  - 매수 잔량 > 매도 잔량 2배: +5점
```

### 테마/재료 (10점)
```python
신규성:
  - 당일 신규 테마: 10점
  - 뉴스/공시 당일: 8점
  - 기존 테마 재부각: 5점
```

## 매매 규칙

### 진입
```
10:00~10:30 진입 권장
- 급등 후 소폭 조정 시 진입
- 조정 없으면 즉시 진입
- 10:30 이후는 신중 (점심 시간 변수)
```

### 익절
```
+5% 도달: 50% 익절
+7% 도달: 100% 익절

또는
- 거래량 급감 시 즉시 청산
- 호가창 역전 시 즉시 청산
```

### 손절
```
-2%: 50% 손절
-3%: 100% 손절 (무조건)

또는
- 5분봉 음봉 전환 시 주시
- 3분봉 2연속 음봉: 청산 고려
```

### 시간 손절
```
14:00: 수익 없으면 청산
15:00: 무조건 전량 청산 (장 마감 리스크)
```

## 리스크 관리

### 포지션 크기
```
1회 매매: 전체 자산의 10% 이하
동시 보유: 최대 3종목
이유: 고위험 전략, 집중 관리 필요
```

### 일일 손실 한도
```
일일 손실 -5% 도달 시:
  → 당일 매매 중단
  → 다음날 재개
```

### 주의 시간대
```
11:30~13:00 - 점심 시간 (변동성 감소)
14:30~15:00 - 외국인 프로그램 매도 (급락 위험)
```
